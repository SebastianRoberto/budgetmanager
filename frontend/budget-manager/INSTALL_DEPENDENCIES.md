# Instalación de Dependencias

## ¿Cómo funciona?

En Angular/Node.js, el archivo `package.json` es el equivalente a `requirements.txt` de Python o `composer.json` de PHP.

## Instalación Automática al Clonar

Cuando un usuario clona el repositorio, solo necesita ejecutar:

```bash
cd frontend/budget-manager
npm install
```

Esto lee el `package.json` e instala automáticamente todas las dependencias listadas.
